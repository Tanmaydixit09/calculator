# Advanced Calculator

An enhanced scientific calculator built with HTML, CSS, and JavaScript.

## Features
- **Basic Arithmetic**: +, -, *, / with operator precedence (PEMDAS)
- **Scientific Functions**: sin, cos, tan (degrees), log10, sqrt, π
- **Memory Functions**: MC, MR, M+, M-
- **Advanced**: Parentheses, keyboard support, calculation history (last 5)
- **UI/UX**: Button animations, mobile responsive, dark/light theme ready, error handling (no eval!)

## Usage
1. Open `index.html` in browser
2. Use mouse or keyboard (Enter=, Esc=AC, Backspace=DEL)
3. History shows recent calculations

## Safe & Secure
- Uses shunting-yard algorithm parser (no `eval()`)
